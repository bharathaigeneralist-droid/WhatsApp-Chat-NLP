📊 WhatsApp Chat Analysis using NLP
📌 Overview

This project analyzes WhatsApp chat data to extract meaningful insights using Natural Language Processing (NLP) and data visualization techniques. It transforms raw, unstructured chat text into structured data and generates insights about user behavior, sentiment, and communication patterns.

🚀 Features
📂 Chat Data Extraction
Converts exported WhatsApp chat text into a structured dataset.
🧹 Data Preprocessing
Cleans text using regex, removes stopwords, and handles timestamps.
😊 Sentiment Analysis
Classifies messages into positive, negative, and neutral using NLP techniques.
☁️ Word Cloud Visualization
Displays frequently used words in a visually appealing format.
😂 Emoji Analysis
Identifies most commonly used emojis in conversations.
📈 Activity Analysis
Daily, weekly, and monthly timelines
Most active users
Message frequency trends
👥 User Insights
Compares participation and behavior among different users.
🛠️ Technologies Used
Programming Language: Python
Libraries:

Matplotlib, Seaborn (Visualization)

WordCloud
Regex
📂 Project Workflow
Import WhatsApp chat data
