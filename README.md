# WhatsApp-Chat-NLP
This project analyzes WhatsApp chat data using NLP and data visualization to extract insights like user activity, sentiment, and communication patterns. It converts unstructured chat text into meaningful information through preprocessing, analysis, and visual representations.

## Dataset used
<a href="https://github.com/bharathaigeneralist-droid/WhatsApp-Chat-NLP/commit/2a6b954e3ba6ac374389ef0257fb54f3162be27d">Data</a>

##  Key Features
- 📊**Automated Chat Parsing** – Extracts structured data (date, time, sender, message) from raw WhatsApp .txt files
- 😊 **Sentiment Analysis** – Uses NLP techniques (VADER) to classify messages as positive, negative, or neutral
- ☁️ **Word Cloud Generation** – Visualizes most frequently used words for quick insights
- 😂 **Emoji Analysis** – Identifies and ranks most-used emojis in conversations
- 📈 **Interactive Visualizations** – Generates graphs for message trends, activity timelines, and user participation
- 🔍 **Text Preprocessing** – Cleans data using tokenization, regex, and stopword removal
- 👥 **User-wise Insights** – Compares activity levels and behavior across participants

  ## Benefits
- Helps understand communication patterns and user behavior
- Useful for social media analysis, customer insights, and relationship trends
- Saves time by converting unstructured chat data into meaningful insights
- Demonstrates ability to work with real-world messy data
- Can be extended for business analytics or chatbot improvement

 ## Novelty
- Combines text mining, sentiment analysis, and visualization in one pipeline
- Handles real-world WhatsApp chat format, which is semi-structured and noisy
- Integrates emoji intelligence, often ignored in traditional NLP projects
- Provides end-to-end analysis: from raw text → preprocessing → insights → visualization
- Practical, relatable dataset (not generic datasets), making it more impactful
